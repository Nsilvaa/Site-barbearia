# GameStorm

🎮 GameStore Landing Page

Uma landing page moderna para loja de jogos digitais, onde os usuários podem visualizar os jogos disponíveis, conferir preços e realizar compras diretamente pelo WhatsApp através do botão “Comprar Agora”.

🚀 Funcionalidades
📌 Exibição de jogos em destaque
💰 Preços visíveis e organizados
🖼️ Cards responsivos para os jogos
📱 Integração direta com WhatsApp
⚡ Design simples, rápido e moderno
🎨 Interface responsiva para desktop e mobile
🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript

📷 Preview

[Preview do Projeto](https://nsilvaa.github.io/GameStorm/)

🎯 Objetivo do Projeto

Este projeto foi desenvolvido para facilitar a venda de jogos digitais através de uma interface simples e direta, usando o WhatsApp como canal principal de atendimento e fechamento de vendas.
